# Facebook & Instagram Changemaker Awards Ethiopia

Independent awards platform for Ethiopian creators.

## Setup
1. Copy `.env.example` to `.env.local`.
2. Add Supabase keys and app URL.
3. Run migration in `supabase/migrations/20260523_init.sql`.
4. Start app with `npm run dev`.
